LIA-Staylo

Una app de corrección pensada para manuscritos completos, no solo textos cortos. Revisa ortografía, gramática, puntuación y estilo con reglas adaptadas a español (México), español (Latinoamérica) e inglés (US).

🪟 Descargar para Windows Ver características

Versión actual: 0.6.x · Requiere Windows 10/11 de 64 bits.

¿Qué hace LIA-Staylo?

Corrección profunda

Analiza tu manuscrito completo y detecta errores de ortografía, gramática y puntuación con la ayuda de LanguageTool y reglas personalizadas.

Ortografía Gramática Puntuación

Enfoque en estilo narrativo

Señala muletillas, repeticiones y frases pesadas para ayudarte a lograr un estilo más claro y profesional, pensado para narrativa y ensayo.

Muletillas Repeticiones Claridad

Diccionario personal

Añade nombres de personajes, lugares, términos técnicos y neologismos de tu universo para que no vuelvan a marcarse como errores.

Universos de ficción Terminología propia

Hecha para autores de habla hispana

Perfiles de idioma

LIA-Staylo trabaja con tres variantes:

  • Español (México)
  • Español (Latinoamérica)
  • Inglés (US) para fragmentos bilingües

Ideal para…

  • Autores independientes que se autopublican
  • Pequeñas editoriales y correctores freelance
  • Creadores de contenido que escriben en español

Preguntas frecuentes

¿LIA-Staylo ya está en versión final?

Actualmente LIA-Staylo se encuentra en fase de pruebas para escritorio (Windows). Las funciones principales de análisis y corrección ya están operando, pero seguimos afinando detalles y mejorando la experiencia.

¿Qué formatos acepta?

Puedes subir manuscritos en .txt, .md y .docx. El texto revisado se puede descargar de nuevo en TXT, Markdown o DOCX.

¿Cómo puedo dar retroalimentación?

Si quieres probar la app, reportar errores o sugerir nuevas funciones, puedes escribir directamente por correo o WhatsApp.
